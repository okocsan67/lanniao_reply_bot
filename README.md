LanNiao X AutoReply 是一款为 X 平台设计的 Chrome 扩展，帮助用户快速生成简短的 AI 回复。用户可在推文详情页点击“AI”按钮，生成评论，评论由 AI 提供支持。所有操作需用户手动触发，符合 X 平台规则。 功能：

在推文详情页添加“AI”按钮。
生成简洁的中文回复。
支持用户配置密钥。
支持自定义提示词。

支持自动跟首页推文互动，设置互动间隔，全自动化，但是有可能触发X风控，自行考虑是否使用！

使用前需要在插件图标处，右键->选项设置你的API key

使用deepseek flash模型，比较稳定，便宜
